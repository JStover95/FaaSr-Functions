# FaaSr-Functions

This repository holds various R and Python functions that can be used as examples/templates for you to start developing your own workflows.

Each folder includes example JSON workflow configuration files, FaaSr-ready R and/or Python functions, and a README describing the workflow and functions.

These examples assume that you have completed going through the [simple FaaSr tutorial](https://faasr.io/FaaSr-Docs/tutorial/) and have setup your FaaSr-workflow repository and secrets according to the instructions there
